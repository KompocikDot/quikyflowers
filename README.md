# QUIKY FLOWERS


#### What is the reason it exists?

##### This tiny API was made especially for needs of small flowerist's which didn't want to use card terminal to accept payments but needed to accept online payments.


## Technologies used:

- Backend
    - `actix-web`
    - `sqlx`
    - `reqwest`

- Databases
    - `postgres`

- DevOps
    - `docker`

- Payment processor
    - `Adyen`
